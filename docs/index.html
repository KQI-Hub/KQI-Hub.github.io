Welcome to the KQI website!

The KQI Hub is a project of Mari's, and the goal is to create simple yet versatile file formats.

The following is a list of formats and systems finished, planned, or in the works:

  • scQOI - A "general-purpose" compression algorithm based on QOI's opcodes.
  • MQIF (.mqif, .qif) - A simple GIF clone based on the scQOI compression algorithm, intended as a proof of concept for scQOI (It can come close to GIF for certain images).
  • KQI (.kqi) - A planned lossy image format based on a few simple concepts.
  • Motion KQI (.mkqi, .mkq) - A form of KQI, conceptually similar to Motion JPEG.
  • KQV - A simple video codec with basic motion vectors.
  • KQC - A simple container format based on RIFF.

Why KQI?
KQI is a project designed to make simple yet effective file formats, while they aren't intended to be as efficient as mainstream formats, they are explicitly made to be simple and fairly easy to implement.

What does KQI stand for/mean?
KQI doesn't necessarily mean anything, but the "I" means image. "KQI" came from "JPG", in the same cypher that applying to "PNG" gives "QOI".
